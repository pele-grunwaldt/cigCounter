Ajustes de sincronización

Para guardar los datos en tu repo de GitHub. Se completa una sola vez y queda guardado en este dispositivo. El token es gratis y solo se guarda acá, en tu celular.

¿Cómo genero el token? (paso a paso)
  1. Andá a GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Tocá Generate new token.
  3. En Repository access elegí Only select repositories → cigcounter.
  4. En Permissions → Repository permissions → Contents poné Read and write.
  5. Generalo, copialo y pegalo abajo. (No se vuelve a mostrar.)
Zona peligrosa
Borra todos los registros en el celular y en GitHub. No se puede deshacer.
